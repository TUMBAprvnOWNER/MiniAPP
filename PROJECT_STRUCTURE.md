## 🎯 Основные компоненты

### Telegram Бот (bot/index.js)
- ✅ Обработка команд /start, /help, /tracker
- ✅ Кнопка запуска мини-приложения
- ✅ API эндпоинты для данных токена
- ✅ API для графика (свечи)
- ✅ API для рыночного стакана

### Мини-приложение (webapp/)
- ✅ Интеграция с Telegram Web App API
- ✅ Отображение тикера и цены
- ✅ Свечной график (lightweight-charts)
- ✅ Рыночный стакан (bids/asks)
- ✅ Кнопки навигации (канал, поддержка, админ)
- ✅ Адаптивный дизайн
- ✅ Поддержка темной темы Telegram

## 🔑 Ключевые технологии

**Backend:**
- Node.js + Express
- node-telegram-bot-api
- CORS для кросс-доменных запросов

**Frontend:**
- Vanilla JavaScript (ES6+)
- Vite (сборщик)
- lightweight-charts (графики)
- @twa-dev/sdk (Telegram Web App)

## 📡 API Endpoints

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `/api/token-data` | GET | Данные токена (цена, объем) |
| `/api/chart-data` | GET | Данные для графика |
| `/api/orderbook` | GET | Рыночный стакан |

## 🎨 Функции мини-приложения

1. **Header**
   - Тикер токена (PRVN)
   - Полное название
   - Текущая цена
   - Изменение цены (%)

2. **График**
   - Свечной график
   - Интерактивный
   - Масштабирование
   - Кроссхейр

3. **Рыночный стакан**
   - Заявки на покупку (Bids)
   - Заявки на продажу (Asks)
   - Цена и объем

4. **Навигация**
   - Кнопка "Канал"
   - Кнопка "Поддержка"
   - Кнопка "Админ"

## 🔄 Автообновление

Данные обновляются каждые 10 секунд:
- Цена токена
- График
- Рыночный стакан

## 🚀 Команды для запуска

```bash
# Установка всех зависимостей
npm run install:all

# Запуск в режиме разработки
npm run dev

# Запуск только бота
npm run dev:bot

# Запуск только веб-приложения
npm run dev:webapp

# Сборка для продакшн
npm run build

# Запуск в продакшн
npm start
```

## ⚙️ Настройка .env

```env
BOT_TOKEN=your_bot_token_here          # От @BotFather
WEBAPP_URL=http://localhost:5173       # URL веб-приложения
PORT=3000                              # Порт API сервера
NODE_ENV=development                   # Режим работы
```
